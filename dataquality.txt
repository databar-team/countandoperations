Model for Company

1.INFORMATION
-Year (int) - Year
-Industry_name_NZSIOC (String) - Name of industry
-Industry_code_NZSIOC (int) - Code for given indrustry

2.PROPERTIES
-Units (string) - EUR/DOLAR
-Variable_code(int) - Code for given Variable
-Variable_name(string) - Variable name
-Variable_category(string) - Category of variable
-Value (int) - Value of variable 

3.TRACKING
-last_save (int) - date when we last save this file
-changes (int) - Flag for changes (1 if it had changes today or 0 if it not)
